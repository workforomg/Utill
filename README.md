## 심박수 삽입기.

심박수 주입기 입니다.

요구사항
```
TamperMonkey가 필요합니다.
PC에 블루투스가 필요합니다.
심박수 측정 기기가 필요합니다. (심박수 측정 웨어러블 기기 or 갤럭시 워치 or 애플 워치) 
```
### 심박수 측정기기
1. 기기와 pc 블루투스 연동

2. 스크립트 실행(크랙에서만 활성화됨.)

3. 오른쪽 하단에 UI로 활성화 -> 왼쪽 상단에 페어링 기기 선택

4. 스챗 채팅창에 **alt+우클릭** 으로 현재 심박수 붙여넣기. (자동으로 \*심박수 : 현재 심박수\*로 붙여넣기 됩니다. 붙여넣고  이어서 써야지, 붙여넣기 한 심박수가 사라지지 않습니다.)

### 갤럭시 워치 
1. 어플 설치 -> BLE Heart Rate Transmitter (유료, 워치에 설치 필요.)

2. 블루투스로 워치와 PC 연결.

3. 스크립트 실행(크랙에서만 활성화됨.)

4. 오른쪽 하단에 UI로 활성화 -> 왼쪽 상단에 페어링 기기 선택

5. 스챗 채팅창에 **alt+우클릭** 으로 현재 심박수 붙여넣기. (자동으로 \*심박수 : 현재 심박수\*로 붙여넣기 됩니다. 붙여넣고  이어서 써야지, 붙여넣기 한 심박수가 사라지지 않습니다.)

### 애플 워치 (워치->아이폰->pc 연동딜레이있음)
1. 어플 설치 -> HeartCast (무료, 워치와 아이폰에 각각 설치)

2. 어플로 워치와 아이폰 연동

3. 아이폰과 PC를 블루투스로 연동

4. 스크립트 실행(크랙에서만 활성화됨.)

5. 오른쪽 하단에 UI로 활성화 -> 왼쪽 상단에 페어링 기기 선택

6. 스챗 채팅창에 **alt+우클릭** 으로 현재 심박수 붙여넣기. (자동으로 \*심박수 : 현재 심박수\*로 붙여넣기 됩니다. 붙여넣고  이어서 써야지, 붙여넣기 한 심박수가 사라지지 않습니다.)

```
# 심박수 (기본=85)
- 수치만 참고하여 출력엔 쓰지 않는다.
```
